Install the ProDocStore extension

The browser extension lets you chat with an AI agent to edit any docs page directly from the rendered site. Two install paths: the Chrome Web Store (recommended for everyone) or a self-host .zip download (for power users who don't want to wait on store review for a patch).

Chrome Web Store (recommended)

One-click install, automatic updates, no developer-mode warning.

Will be listed as unlisted - reachable only via this link, not via store search. Until the listing is live, use the .zip download below.

Compatible with any Chromium browser that supports Manifest V3 and the chrome.sidePanel API: Chrome 120+, Edge, Brave, Arc.

Download .zip (power users)

Use this when you need a same-day patch that hasn't cleared store review, or if you don't want to depend on the Chrome Web Store.

Download latest release

Pick the prodocstore-v*.zip asset on the latest release. The zip contains a ready-to-load dist/ folder.

Load the .zip into Chrome

Skip this section if you used the Chrome Web Store path.

  1. Unzip the downloaded prodocstore-v*.zip. You get a folder with manifest.json at the top level.
  2. Open chrome://extensions.
  3. Toggle Developer mode (top-right).
  4. Click Load unpacked.
  5. Pick the unzipped folder.
  6. Pin the extension to your toolbar so the icon is always visible.

Heads up: Chrome shows a yellow "disable developer-mode extensions" banner every browser launch when you have unpacked extensions installed. The Chrome Web Store path avoids that. If the banner is annoying, switch to the store install once it's published.

After install

  1. Open any ProDocStore-enabled docs site.
  2. Click the ProDocStore toolbar icon. Chrome opens a side panel.
  3. The banner under the header shows the repo and source path resolved from the URL.
  4. Click the gear icon to open the options page and sign in with GitHub (one-time device-flow auth).
  5. Pick a backend (Claude, OpenAI) and start chatting.

For backend setup, auth, and what each adapter does, see Extension.

Updating

Chrome Web Store install: Chrome auto-updates the extension in the background. Nothing to do.

.zip install: download the new release, replace the unzipped folder in place, and click the Reload arrow on the ProDocStore card in chrome://extensions.
