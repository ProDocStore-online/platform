Microphone & dictation

Click the mic icon in the chat to dictate prompts instead of typing. Chrome has a quirk that makes this confusing on first use:

  • Microphone permission is granted per origin.
  • The mic icon in the address bar grants the page (e.g. *.pages.dev) - that does not cover the side panel.
  • The side panel runs in the extension's own origin (chrome-extension://…) and needs a separate Allow.
  • Chrome's normal "Allow" prompt sometimes does not appear inside side panels. The fix is a one-time visit to the extension's microphone setting.

Set Microphone to Allow on the page that opens, then click the mic icon in the chat again. Once granted, Chrome remembers - you only do this once.
