# True Non-Profit

FreeDocStore's first public docs book.

This book defines a first-principles body of knowledge for non-profits and an assessment model for the "trueness" of non-profit organizations.

## Source

Published pages live in `docs/`.

When the platform GitHub Pages workflow runs, this folder is published at:

```text
/books/true-non-profit/
```

## Current Status

Draft 0.1:

- public register scaffold
